# Java Logging using SLF4J and Logback Appenders

This project demonstrates error level logging, warning level logging, and parameterized logging using SLF4J and Logback Classic framework.

## Implemented Exercises
1. **Logging Error & Warn**: Standard error and warn logs using SLF4J API in `LoggingExample`.
2. **Parameterized Logging**: Logging custom runtime variables without string concatenation overhead.
3. **Logback Configuration Appenders**: Logging to both Console (using `ConsoleAppender`) and a local file named `app.log` (using `FileAppender`).

## How to Run Tests
Execute the following Maven command:
```bash
mvn clean test
```
